1:删除文件及文件夹: rm -rf 目录名称
2:查看linux下的java进程：　ps -ef | grep java  或者 ps auxf | grep java
3.杀死JAVA进程： 【pkill java】 （在有反串改的时候可以用，杀死所有java进程）  或者【 kill -9 进程ID】
4.复制文件　cp  -r  /user/oa  /usr_back_oa
5.在linux下修改文件内容
　　1)选用vi选择需要修改的文件vi index.jsp
　　2)然后会出现提示内容按一下i 进行修改
　　3)修改完成后按esc键后再按 shift+冒号 最后输入wq

6.查看硬盘是否有满: df -h
7.压缩文件： zip -r ./project.zip ./目录名 -r         【./project】:要压缩成的文件  【./目录名】:表示要压缩的目标  【-r】:表示递归
8.解压缩：解压 unzip xahot.zip 不解释    
8.1  解药到指定的目录： unzip  ghost.zip  -d  /opt/www/ghost
8.2 解压tar.gz文件    tar xzf redis-2.8.3.tar.gz 解压到指定的位置：tar xzf redis-3.0.3.tar.gz -C ../runfile_redis/

9.修改mysql数据库不区分大小写文件  修改、/etc/my.cnf，在[mysqld]后面加上lower_case_table_names=1，重启mysql服务即可。
10.查找文件   find / -name "my.cnf"     "/"表示根目录  如果要在跟目录里面查找对应的文文  可以在这样: /pagageName
11.实时查看tomcat日志文件 tail -f -n 500 catalina.out  //-n 500 可以忽略是为了显示多少行 
12.创建文件夹命令root>mkdir [folderName]  ,创建文件root>touch  /a.txt
13.查看内存信息  jmap -histo  12538 > xx.txt       解释12538是进程号
14.vi开启文件强制退出后，进入会有备份。可以执行  >rm -rf  .*.swp
15.查找40124端口占用的进程  >netstat -tupln |grep 40124  
16.查看进程号3131占用的信息 lsof -i|grep 3131
17.下载文件   wget  http://download.oracle.com/otn-pub/java/jdk/8u141-b15/336fa29ff2bb4ef291e347e091f7f4a7/jdk-8u141-linux-x64.tar.gz
18.安装rz  sz 命令   yum install lrzsz
19.安装c++编译环境  yum install gcc-c++
