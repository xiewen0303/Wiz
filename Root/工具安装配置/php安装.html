1. 【下载php源码】：从php的官方网站下载php的安装源码包。

  2. 【解压安装包】：下载的安装包是经过压缩的gz格式，在linux可以使用自带的工具tar进行解压，在安装包所在目录执行命令：tar -zxvf php-5.5.11.tar.gz

  3. 【配置安装变量】：打开解压后的目录-执行命令：cd php-5.5.11；配置安装环境-执行命令：./configure --prefix=/usr/local/servers/php ，“/usr/local/servers/php”是安装路径，可以改成自己喜欢的安装路径。

  4. 【编译源码】：在解压目录执行编译命令：make

  5. 【安装php】：切换到root用户-执行命令：sudo su root；安装-执行命令：make install

    创建软链：执行命令：ln -s /usr/local/servers/php/bin/php /usr/local/bin/php
